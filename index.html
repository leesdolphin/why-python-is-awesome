Asynchronous coding

Async?

Usually in programming you do one thing then the next then the next.
But what if you COULD do multiple things at once? That would make your program faster; but harder to understand.

Threadding

In most languages(Python's kinda an outlier with it's GIL) threads allow multiple lines of execution to run at once.
The problem is that you have to be careful. What if you many threads are trying to insert into an array?

Multiple Processes

Python's got multiprocessing. But sharing data becomes harder; and you have to manage sending/recieving data.

Async - AsyncIO/Single Threaded Asynchronous

It's a single thread in a single process so no locking/unlocking or sharing problems.
Plus the developer controls when the switches happen.
This only really is useful for IO bound operations - like a server, or a web crawler.

Generators

Python generators make simple stuff simple and hard stuff quite simple too. Plus with the new yield from you can make generators that compose other generators without having to loop over them yourself. It's great. 

def fib():
  next, curr = 1, 1
  while True:
    yield curr
    next, curr = curr + next, next

## This prints a lot of numbers very quickly.
for fib_num in fib():
  print(fib_num)
            

function* fib(){
  var next = 1, curr = 1, swap;
  while(true) {
    yield curr;
    swap = next;
    next = curr + next;
    curr = swap;
  }
}
// Also prints a lot of numbers very quickly.
for(var fib_num of fib()) {
  console.log(fib_num);
}

Writing them is also really easy 

from contextlib import contextmanager

@contextmanager
def exit_on_error():
  try:
    yield  ## `with` block is executed here.
  except Exception:  ## `with` block raised an error.
    import sys
    sys.exit(1)

print("Hello, I'm going to divide by zero")
with exit_on_error():
  one_over_zero = 1 / 0
  ## Python exits instead of printing a stack trace.

Monkey Patching

You can 'fix' library code without needing to subclass or worse, edit the file.
And you can add if statements into class definitions - who knew this was a thing. 

import sys

class MyCoolClass():

  if sys.version_info < (3, 0):
    def __unicode__(self):
      return "Some unicode representation"

  def __str__(self):
    return "Some string representation"
